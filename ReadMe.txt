# Student Performance Dashboard (ETL Pipeline)

This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python and Streamlit. It reads student performance data, processes it using Pandas, and presents key insights on an interactive web dashboard.

## Features

- Loads data from CSV (Extract)
- Cleans and summarizes student data (Transform)
- Displays:
  - Average marks per course
  - Student distribution
  - Full student list (Load)

## 🛠️ Built With

- Python
- Pandas
- Streamlit
- Matplotlib
